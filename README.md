## Description

This project is a simple Python program that evaluates student attendance.

It is a basic script that analyzes the number of classes attended.  
The program is a simple example of conditional logic in Python.


## How it works

1. The program asks the user to enter the number of classes attended.
2. The system converts the value into an integer.
3. The program checks the number using conditional statements.
4. If the number is less than five, it shows low attendance.
5. If the number is between five and eight, it shows medium attendance.
6. If the number is nine or more, it shows high attendance.

The program uses simple conditions to determine the attendance level.

Currently, the system is processing the user input and evaluating the attendance category.
